The three phases
Phase 1
Movement Daily
One metric: did you move today? Gym / treadmill / floor. Wake-up time doesn't count yet.
Am I ready for Phase 2?
Ready when: movement days clearly outnumber rest days most weeks / moving feels like the default not the decision / rest feels like recovery not relief / true for at least 3 weeks in a row.
Not ready if: alternating movement and rest most days / still deciding each morning whether to move / good weeks followed by collapse weeks.
Phase 2
Wake-up Anchor
Once daily movement is automatic — add one consistent morning trigger. A cue, not a time.
Phase 3
Full Routine
Wabi and Sunpo fold in naturally. You're already up and moving.
What "showed up" means
Full: Gym on gym days / treadmill on others
Floor: Any movement, any duration, first floor
Rest: One day. Logged. Not a failure.
Two rest days → floor rule on day three.
You already became the gym person.
You're becoming the daily movement person.
Same process.
You don't build discipline and then show up.
You show up imperfectly and discipline is what that becomes.
